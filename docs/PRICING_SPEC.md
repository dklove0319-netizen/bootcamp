[절대 규칙]
1. AI 출력에 유형·성격 라벨 금지. 출력은 항상 [사용자 기록 직접 인용 + 사실/망상 측정값 + 미완결 질문] 형식. 마지막은 반드시 질문으로 끝남. (2026-07-05 용어 통일: 전 화면 "망상")
2. 사용자가 실제 입력한 기록만 인용. 창작·각색 절대 금지.
3. 데이터 연구 활용은 명시적 동의로만. 동의 체크박스 기본값은 해제.
4. 가격은 하드코딩 금지, 설정으로 관리 (추후 49,000 → 79,000 → 120,000 인상 — 인상은 시간이 아니라 근거(후기·완주율·전환율)로).

[가격 구조]
- 무료: 즉시 측정. 기록 1건 → 분석 1회. 비로그인 1일 1회, 로그인 1일 3회. 결과 저장만 가입 필요.
- 유료: 21일 검증 코스 49,000원 (2026-07-05 개정: 34,000 → 49,000 — 검증 국면의 표본 질 필터 + 정가 대비 할인율 신뢰(59%) 확보). 화면에 정가 120,000원 취소선 + "지금 참여자의 기록은 오제로의 효과 검증에 쓰여요. 그 동의가 이 가격의 조건이에요." 표시. 21일 거울 보고서 포함.
- 3일 상품은 만들지 않음. 대신 환불 규칙: 결제 후 72시간 이내 + 기록 3건 이하면 전액 환불. 환불자 데이터는 연구셋에서 제외. 환불률·사유는 관리자 지표로 집계.

[동의 처리]
- consents 테이블: user_id, 문구 버전, granted_at, revoked_at. 철회 가능(철회 후 코스 이용은 유지).
- 동의 문구: "내 기록이 익명 처리되어 오제로의 패턴 연구와 효과 검증에 활용되는 것에 동의해요. 언제든 철회할 수 있어요."
- 연구용 데이터 추출 쿼리는 동의 유효 + 환불 제외 조건 필수.

[21일 구조]
- 하루 1회 기록+분석, 미리 당겨쓰기 불가. 완주 기준: 17일 이상 제출(설정값).

[공유 카드]
- 측정값만 노출(예: "사실 2 : 망상 7"). 기록 원문 절대 미포함. 링크는 측정 시작 화면으로.

[카피 톤]
- ~이에요/~해요 체. 위로 문구·이모지·영어 외래어·조언 금지. 짧은 평서문.
