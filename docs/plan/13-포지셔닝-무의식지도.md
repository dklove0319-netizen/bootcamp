# 포지셔닝 — 무의식을 관찰 가능한 데이터로 바꾸는 시스템 (운영자 제공 2026-07-10)

> 운영자가 붙여넣은 포지셔닝 문서의 채택 요지 + 기준 대조 교정 4개. 원문 전문은 대화 기록에.

## 채택 (기준과 일치)
- **본질**: 생각을 고치는 게 아니라, 생각이 만들어지는 구조를 보여주는 제품. "사람은 생각 때문에 힘든 게 아니라 자신의 무의식 패턴을 보지 못해 같은 현실을 반복한다."
- **진짜 문제**: 불안·우울(증상)이 아니라 "알면서도 같은 생각·감정·선택을 반복하는 문제" — 후기 분석(docs/plan/11)과 일치.
- **해자**: AI가 아니라 **개인의 무의식 패턴 데이터셋** — 구조화 저장(눈금·구별·잇기·문답 전부 별도 칸) + 금고가 이미 그 데이터셋.
- **한 문장**: "반복되는 인생을 바꾸려는 서비스가 아니라, 반복이 왜 만들어지는지를 처음으로 '보이게' 만드는 자기관찰 시스템."

## 교정 4개 (기준 대조 — 2026-07-10 판정)
1. "사실/망상을 **자동으로** 분리" ✕ → 사용자가 먼저 구별, AI는 대조 (절대 원칙 — AI가 먼저 답하면 훈련이 의존이 된다).
2. 예시 질문 "그 믿음은 언제부터 생겼나요?" ✕ → 과거 심입은 심화 35 영역 (앱은 현재 글 안의 전제만 — 퍼널 역할 분담). "그게 정말 사실인가?" ✕ → 네/아니오 닫힘 질문 (질문 닫기 규칙 위반).
3. "변화 **예측**" ✕ → 예언은 이 앱의 정의상 망상. 해자는 예측이 아니라 관찰의 정밀도.
4. "반복**사람**" 축 ✕ → 사람을 지도에 올리면 상대 판결(금지). 지도의 축은 사람이 아니라 그 사람 위에 붙인 문장.

## 기능 6개 대조 (2026-07-10 시점)
기록 ✅ / 분리 ✅(사용자 먼저) / 질문 ✅(블라인드 동률) / 21일 패턴 분석 ✅ / **무의식 지도(한 장) ⚠️ 유일 공백 → M-1로 빌드** / 변화 추적 ✅("반응속도 초 단위"는 측정 불가 — 기록된 회수 동작이 그 증거).
