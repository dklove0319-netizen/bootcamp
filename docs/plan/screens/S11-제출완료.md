# S11. 제출 완료 — 동료 존재감 (커뮤니티 기능 없이)

## 들어오는 길
- S10 오늘 기록 마치기 → 자동 진행. **제출한 사람만, 제출 후에만** 도달 (제출 전 타인 콘텐츠 노출 금지 — 지시서 6번)

## 화면 구성 (위에서 아래로)
1. 완료 한 줄: "오늘 기록이 끝났어요." (축하·칭찬 문구 금지)
2. 동료 존재감 한 줄: "오늘 N명의 관찰자가 함께 기록했습니다" (달력 날짜 기준 카운트)
3. 익명 답변 1개: 관찰자 코드(예: o024)와 답변 원문 — 같은 달력 날짜의 공개 동의된 답변 중 무작위 1개를 서버가 선별 (결정 2026-07-05)
4. 홈으로 버튼

## 규칙
- 익명 답변은 클라이언트가 표를 직접 읽지 않고 서버가 1개만 골라 내려준다 (05-db.md 공유 층).
- 댓글·반응·공유 버튼 없음. 읽기만.

## 행동과 흐름
| 행동 | 성공하면 | 실패하면 |
|---|---|---|
| (진입 시) N명·답변 요청 | 표시 | 실패 시 이 블록 자체를 숨김 (완료 화면은 유지 — 루프에 지장 없음) |
| 홈으로 | S01 | — |

## 예외 5종
| 상황 | 처리 |
|---|---|
| 비어 있을 때 (공개 답변 0건) | 답변 칸 없이 N명 문구만. N=1(나뿐)이면 "오늘은 당신이 기록했습니다." |
| 로딩 중 | 해당 블록 스켈레톤 |
| 실패 | 블록 숨김 (위) |

| 잘못된 입력 | 해당 없음 |
