# S02. 오프닝 — 어제의 회수 (하루 루프 단계 1)

## 들어오는 길
- S01 홈에서 "오늘 기록 시작" (서버가 열림 판정 통과 시에만)

## 화면 구성 (위에서 아래로)
1. 어제 문장 원문 인용 (날짜 포함, 분석·해석 없이 인용만 — 예: "7월 4일에 이렇게 쓰셨어요 — '(원문)'")
2. 어제 정한 작은 행동 + 실행 여부 선택: [했다 / 부분 / 안 했다] — 평가 문구 없음
3. 다음 버튼

## 규칙 (지시서 3-1)
- 어제 기록이 없으면 인용하지 않는다. 대신 복귀 문구: "어제는 기록이 없는 날입니다. 안 쓴 날은 어떤 날이었나요." (연속 기록 깨짐 표시 금지)
- 인용 문장은 서버가 어제 free_text에서 고르고 원문 일치 검증 후 내려준다.

## 행동과 흐름
| 행동 | 성공하면 | 실패하면 |
|---|---|---|
| 실행 여부 선택 | 어제 entry의 action_result 저장 | "저장하지 못했어요. 다시 시도해주세요." |
| 다음 | S03 상태 체크 | 위와 동일 |

## 예외 5종
| 상황 | 처리 |
|---|---|
| 비어 있을 때 (어제 기록 없음 / 1일차) | 인용·행동 칸 없이 복귀 문구(위) 또는 1일차 안내: "오늘이 1일차예요." |
| 로딩 중 | 인용 자리 스켈레톤 |
| 실패 (통신) | "어제 기록을 불러오지 못했어요." + 재시도 (그래도 실패 시 인용 없이 진행 허용) |
| 권한 없음 | S14로 |
| 잘못된 입력 | 해당 없음 (선택뿐) |
| (추가) 어제 행동을 안 정했던 날 | 실행 여부 칸 자체를 노출하지 않음 |
