# S12. 중간 거울 (7·14일째) + 시차 대면 (14일째)

## 들어오는 길
- 7일째·14일째에 S01 홈에 항목 노출 → 누르면 진입 (하루 루프와 별개 — 루프를 막지 않음)

## 화면 구성 (위에서 아래로)
1. 사실 제시 1~3줄: 예 "지금까지 기록에서 가장 많이 반복된 단어는 '___', '___', '___'이에요." / "가장 자주 이름 붙인 감정은 불안(5번), 짜증(3번)이에요." — 해석 없음 (단어·감정 빈도는 서버 계산, 08-ai-prompts.md 3번)
2. (14일째만) 시차 대면: "1일째 당신의 답변이에요." + 답변 전문 그대로 + 날짜
3. 닫기 버튼

## 규칙 (지시서 4번)
- 분석 보고서가 아니다. 짧은 사실 제시.
- 시차 대면은 요약 없이 전문 그대로.
