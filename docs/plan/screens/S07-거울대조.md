# S07. AI 거울 대조 (하루 루프 단계 6 — 채점 아님)

## 들어오는 길
- S06 구별 제출 → 자동 진행 (AI 대조는 서버에서 실행)

## 화면 구성 (위에서 아래로)
1. 안내 한 줄: "당신은 이렇게 나눴어요. 이렇게도 나뉩니다."
2. 나란히 두 열: 내 구별 / AI의 구별 (같은 원문 조각 기준)
3. 다음 버튼

## 규칙 (지시서 2·3번 + 공통 판독 규칙)
- 점수·일치율·정답 표시·색깔로 맞고 틀림 구분 — 전부 금지. 나란히 놓기만 한다.
- 사용자가 사실로 놓은 것을 AI가 망상으로 본 경우: "이 문장은 이렇게도 나뉩니다" 형태로만. AI의 구별도 하나의 거울이지 정답이 아니라는 태도 유지.
- AI가 unclear로 둔 성분은 별도 줄로: "이 문장은 어느 쪽에 놓으시겠어요?" — 사용자가 놓으면 user_split에 반영.
- 용어는 "망상" 그대로 표기 ("해석"으로 순화 금지 — 판독 규칙의 용어 규칙).
- AI 성분의 src는 서버가 원문 일치 검증 후 내려준다. 검증 실패 성분은 버린다. 내부 범주(category)는 화면에 절대 노출하지 않는다.
- 프롬프트는 서버에만 (08-ai-prompts.md 공통 판독 규칙 + 1번).

|---|---|---|

| 다음 | S08 망상·감정 연결 | — |
