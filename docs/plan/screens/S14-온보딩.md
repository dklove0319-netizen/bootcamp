# S14. 온보딩 — 로그인·동의·시각 (블럭 5)

## 들어오는 길
- 비로그인 상태로 어떤 화면이든 접근 시 · 첫 방문

## 화면 구성 (위에서 아래로)
1. 서비스 한 줄 소개 (과장 없이): "기록에서 사실과 망상을 나누는 21일 훈련이에요."
2. 로그인 — 이메일 매직링크 (비밀번호 없음 · 결정 2026-07-05)
3. (첫 로그인만) 동의 2종 체크: (a) 심리 기록의 AI 전송·분석 (c) 익명화 기록의 서비스 개선 활용 — 각각 문서 링크 + 시각·버전 기록. (b) 답변 공개는 매일 건별(S09)이라 여기 없음
4. (첫 로그인만) 기록 시각 정하기: "하루 기록이 열리는 시각이에요." 기본 제안: 21시
5. 시작 버튼 → S01

## 규칙 (지시서 7·8번)
- 초기엔 관리자가 수동 등록한 이메일만 로그인 가능 (결제 없음).
- 동의 (a)는 필수 (거부 시 서비스 이용 불가 — AI 처리가 핵심 기능). (c)는 선택.
- 관리자 열람 사실이 동의 문서에 명시돼야 함 (PRD 6절 약관 항목).

## 행동과 흐름
| 행동 | 성공하면 | 실패하면 |
|---|---|---|
| 로그인 | 등록된 계정 → 진행 | 미등록: "아직 등록되지 않은 이메일이에요. 안내받은 이메일로 시도해주세요." |
| 동의 + 시작 | consents 기록, profiles 생성 → S01 | "저장하지 못했어요" + 재시도 |
