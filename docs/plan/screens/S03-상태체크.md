# S03. 오늘의 눈금 (하루 루프 단계 2 — 10초 안에 끝나는 자기 관찰)

## 들어오는 길
- S02 오프닝 → 다음

## 화면 구성 (위에서 아래로)
1. (가입 후 첫날 딱 1회만) 안내 한 줄: "오늘의 나를 네 눈금으로 남겨요. 이 눈금은 추이가 되어 당신에게 돌아와요." — 이후엔 표시 안 함
2. 기분 — 라벨 "오늘 하루에 깔려 있던 기분" · 0 매우 나쁨 ~ 10 매우 좋음
3. 감정 — 라벨 "오늘 가장 컸던 감정" · 이름 선택(기쁨 · 평온 · 설렘/기대 · 뿌듯함 · 불안 · 짜증/화 · 슬픔 · 부끄러움 · 무덤덤 · 잘 모르겠음) + 세기 0 거의 없음 ~ 10 매우 강함
4. 체력 — 라벨 "지금 몸에 남아 있는 힘" · 0 방전 ~ 10 가득
5. 수면 — 라벨 "지난밤의 잠" · 0 매우 나빴다 ~ 10 매우 좋았다
6. 다음 버튼 (4눈금 + 감정 이름 모두 입력해야 활성)

## 규칙 (측정 일관성 — 21일 내내 동일해야 추이가 성립)
- 라벨은 질문형("어땠나요?")이 아니라 관찰형 — 캐묻기가 아니라 내가 눈금을 남기는 구조.
- 문구·순서·척도·감정 이름 목록은 21일 동안 절대 변경 금지.
- 기본값 없음 — 슬라이더를 미리 5에 놓지 않는다 (앵커링 방지). 손대야 값이 생김.
- 진행 카운터("2/4") 없음. 한 화면.
- 감정 이름 목록은 긍정 4 : 부정 4 균형 (관점 원칙 — 좋은 날도 같은 정밀도로 관찰).
- 이 숫자·이름은 단계 7(망상·감정 잇기)과 중간 거울·보고서에서 회수된다.

## 행동과 흐름
| 행동 | 성공하면 | 실패하면 |
|---|---|---|
| 다음 | 눈금 4종 + 감정 이름 저장 → S04 자유 기록 | "저장하지 못했어요. 다시 시도해주세요." (입력 보존) |

## 예외 5종
| 상황 | 처리 |
|---|---|
| 비어 있을 때 | 해당 없음 (입력 화면) |
| 로딩 중 | 저장 중 버튼 비활성 + "저장하고 있어요" |
| 실패 (통신) | 문구 + 재시도, 입력값 화면 유지 |
| 권한 없음 | S14로 |
| 잘못된 입력 | 미입력 눈금 테두리 표시 + "네 눈금을 모두 남겨주세요" (범위 밖 값은 UI상 불가능) |
