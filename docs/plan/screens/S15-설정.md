# S15. 설정 — 시각·알림·동의·삭제

## 들어오는 길
- S01 홈의 설정 항목

## 화면 구성 (위에서 아래로)
1. 기록 시각: 현재 값 표시 + 변경 — 허용하되 다음날부터 적용 (결정 2026-07-05)
2. 낮 행동 알림: 켜둔 리마인더 확인·끄기
3. 이야기할 수 있는 곳: 자살예방 상담전화 109 (24시간) — 상시 항목 (S05 B안과 짝)
4. 동의 내역: (a)(c) 동의의 시각·버전 열람 + 약관·개인정보처리방침 링크
5. 계정·데이터 삭제 (화면 맨 아래)

## 규칙 (지시서 7번)
- 삭제는 완전 삭제 흐름: 안내("모든 기록이 완전히 지워지고 되돌릴 수 없어요. 공개했던 답변도 함께 사라져요.") → 확인 입력(예: '삭제합니다' 타이핑) → 전 표 실삭제.
- "일괄 공개" 같은 공유 설정은 여기 만들지 않는다 (건별 토글만 — 지시서 6번).

## 행동과 흐름
| 행동 | 성공하면 | 실패하면 |
|---|---|---|
| 시각 변경 | 저장 + "내일부터 적용돼요" | "저장하지 못했어요" + 재시도 |
| 삭제 확인 | 전 데이터 실삭제 → 로그아웃 → S14 | "삭제하지 못했어요. 다시 시도해주세요." (부분 삭제 상태로 남기지 않음 — 서버에서 한 단위로) |

## 예외 5종
| 상황 | 처리 |

| 실패 | 문구 + 재시도 |
