# S16. 시작점 남기기 — 검증된 척도 2종 (첫 기록 직전 1회 + 21일째 1회)

## 들어오는 길
- 생애 첫 "오늘 기록 시작" 탭 직후: S01 → S16 → S02 (이후엔 다시 안 나옴)
- 21일째: S13 보고서 열기 직전 1회 ("시작점과 나란히 놓기 전, 지금의 나를 남겨요.")

## 화면 구성 (위에서 아래로)
1. 안내 한 줄 (첫 회): "첫 기록 전에, 지금의 나를 남겨요. 21일째의 당신이 이걸 다시 봐요." — 검사가 아니라 미래의 나에게 남기는 기록이라는 프레임
2. WHO-5 웰빙 지수 — 5문항 (전부 긍정 진술형)
3. 탈중심화 척도 (경험 질문지 EQ 계열) — 11문항 (능력 진술형: "나는 생각을 생각으로 볼 수 있다" 류)
4. 남기기 버튼

## 규칙
- **척도 문항의 원문과 응답 방식은 절대 수정 금지** — 검증된 척도는 문구를 바꾸면 공인 줄자가 아니게 된다. 우리 톤은 앞뒤 안내 문구까지만.
- 결과 점수를 그 자리에서 보여주지 않는다 (채점 화면이 되지 않게). 회수는 21일 보고서에서 "시작점 / 21일째" 나란히 — 해석 없는 사실 제시.
- 빌드 전 확인: WHO-5 자유 사용 / 탈중심화 척도 한국어판 사용 조건·출처 표기 확인 (미확인 상태로 빌드 금지).
- 동의 연결: 연구·효과 입증 활용은 동의 (c) 문구에 명시.
