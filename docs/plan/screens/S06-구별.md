# S06. 사실·망상 구별 (하루 루프 단계 5 — 사용자가 AI보다 먼저. 이 앱의 핵심 순서)

## 들어오는 길
- S04 자유 기록 → 위기 판정 통과 후 자동 진행

## 화면 구성 (위에서 아래로)
1. 방금 쓴 기록을 조각 단위로 표시 — 시스템이 기계적으로 조각냄 (문장 끝·연결어미 "~고/~서/~는데" 경계. 판단 없는 문법 기준 — AI 분석 아님)
2. 조각마다 태깅: [사실 / 망상 / 해당 없음]
3. 쪼개기: 긴 조각을 탭하면 더 잘게 나눌 수 있음 (혼합 문장 대응 — 어디서 나눌지는 사용자가 정함)
4. 다음 버튼

## 규칙 (지시서 3번)
- **AI의 구별 결과는 이 화면에서 어떤 형태로도 미리 보이지 않는다.** AI가 먼저 답을 주면 훈련이 아니라 의존이 된다.
- 안내는 정의만 (공통 판독 규칙과 동일한 자): "사실 = 카메라와 녹음기에 찍히는 것. 망상 = 그 위에 마음이 붙인 모든 것. 감정이 일어난 것은 사실, 감정의 해석은 망상."
- 용어는 "망상" 그대로 표기 — "해석"으로 순화 금지 (판독 규칙의 용어 규칙).
- 제출하면 이전 단계로 못 돌아간다.

## 행동과 흐름
| 행동 | 성공하면 | 실패하면 |
|---|---|---|
| 문장 태깅 | 화면에 즉시 반영 | — |
| 다음 | user_split 저장 → S07 AI 대조 | "저장하지 못했어요. 다시 시도해주세요." (태깅 상태 보존) |
