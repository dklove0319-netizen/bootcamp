# S01. 홈 — 21일 격자 달력 (오늘 위치 확인과 진입)

## 들어오는 길
- 로그인 직후 기본 화면 · 하루 루프 완료 후(S11 → 홈) · 푸시 알림 누르면 여기로

## 화면 구성 (위에서 아래로)
1. 서비스명 표시 → 2. 21칸 격자(기록한 날만 채움 — 사실의 표시) → 3. 오늘 기록 진입 영역 → 4. 중간 거울·보고서 항목(7·14·21일째만 노출)

## 금지 (지시서 4·11번)
- 배지·포인트·불꽃·연속기록 카운터·축하 애니메이션 절대 금지. 빠진 날을 강조하는 표시(빨간 X 등)도 금지 — 그냥 빈 칸.

## 행동과 흐름
| 행동 | 성공하면 | 실패하면 |
|---|---|---|
| 오늘 기록 시작 | 서버가 열림 판정(클라이언트 시간 판정 금지) → S02 | 안 열림: "오늘 기록은 HH시에 열려요" (버튼 비활성) |
| 이미 제출한 날 다시 누름 | 그날 기록 열람 (읽기 전용) | — |
| 과거 날짜 칸 | 그날 기록 열람 (본인 것만) | 기록 없는 날: "기록이 없는 날이에요" |
| 중간 진행 중 재접속 | last_step 위치로 이어가기 | — |

## 예외 5종
| 상황 | 처리 (문구까지) |
|---|---|
| 비어 있을 때 (1일차 이전) | 격자 전부 빈 칸 + "첫 기록이 1일차가 돼요. 오늘 HH시에 열려요." (1일차 = 첫 기록 제출일, 첫 시작 탭 시 S16 시작점 먼저) |
| 로딩 중 | 격자 스켈레톤 (문구 없음) |
| 실패 (통신) | "불러오지 못했어요. 다시 시도해주세요." + 재시도 버튼 |
| 권한 없음·비로그인 | S14 온보딩으로 이동 (블럭 5부터) |
| 잘못된 입력 | 해당 없음 (입력 없는 화면) |
