## 들어오는 길
- 21일째(이후 상시) S01 홈에 항목 노출 → 진입

## 화면 구성 (위에서 아래로 — 7개 구성, 결정 2026-07-05)
1. 기록 사실 요약: "기록한 날 N일. 기록 없는 날 M일." (평가 없음)
2. 눈금 4종의 날짜별 추이 (표 또는 단순 선 — 서버 계산)
3. 가장 자주 이름 붙인 감정 + 가장 반복된 단어들 (사실만)
4. 1·7·14·21일 답변 전문 나란히 (날짜 포함, day_no 태깅으로 조회)
5. 사용자가 이은 망상·감정 연결 목록 (원문 그대로)
6. 시작점 / 21일째 설문(WHO-5·탈중심화) 숫자 나란히 — 해석·평가 없이 "당신이 남긴 시작점이에요" (진입 직전 S16에서 21일째 설문을 먼저 남김)
7. 마지막 되묻는 질문 하나 (AI — 원문 인용 기반)

## 규칙 (지시서 4·11번)
- 원문 인용 중심. 패턴 명명·유형 라벨·성장 평가·축하 전부 금지.
- 숫자 집계는 서버 계산, AI는 인용 선별과 마지막 질문만 (08-ai-prompts.md 4번).
