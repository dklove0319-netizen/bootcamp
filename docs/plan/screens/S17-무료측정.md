# S17. 무료 측정 — 즉시 측정 흐름 (스펙 1단계 · 비로그인 가능)

> 근거: docs/PRICING_SPEC.md — 기록 1건 → 분석 1회. 비로그인 1일 1회, 로그인 1일 3회. 결과 저장만 가입 필요.
> 용어: 전 화면 "망상"으로 통일 (2026-07-05 사용자 확정 — 이층 가정 폐기, 06-decisions.md).

## 들어오는 길
- 홈(측정 입구) · 공유 카드의 링크 (블럭 6)

## 흐름 (화면 4개가 한 줄로 — 2026-07-05 개정: 사용자 구별이 AI보다 먼저)
1. **입구(홈)**: 서브 "무의식 패턴 분석" + 메인 "오제로의 거울"·"Ozero's Mirror" + "지금 비춰보기" (가격·가입 요구 없음)
2. **기록 입력**: 쓰기 가이드(고정 문안) + 여러 줄 입력칸 + "다 썼어요"
3. **사용자 구별 (신설)**: 기록을 판단 없는 문법 경계(문장 끝·쉼표)로 조각내 표시. 상단에 고정 정의 문구(08-ai-prompts "사용자 노출용 정의 문구") + "사실 · 망상을 구별해보세요." 조각마다 [사실/망상] 태깅 — 전부 나눠야 다음으로. AI 힌트는 이 화면에 절대 없음
4. **결과 (거울 대조 — 2026-07-07 강화)**:
   - "당신은 사실 a : 망상 b 로 나눴어요." (사용자 구별 회수 — 채점·일치율 아님)
   - **조각별 나란히**: 각 조각에 [내 구별 vs 거울] 표시. 어긋난 조각엔 카메라 기준 한 줄 이유(예: "'헤어지자는 소리겠지'는 추측이에요 — 카메라엔 물음표만 찍혀요"). 같게 본 조각엔 코멘트 없음(자세규정 8-6). 혼합 조각은 이유에서 성분을 가른다. "틀렸다/맞았다" 금지 — 거울의 구별도 정답이 아니라 또 하나의 거울
   - 거울 측정값 ("거울은 사실 x : 망상 y 로 나눕니다")
   - 질문은 가장 크게 어긋난 조각 하나를 조준 ('왜' 시작 금지, 답 후보 금지) · 유형·성격 라벨 금지 · 위로·조언 금지

## 규칙
- AI 프롬프트는 서버에만.
- **저장 (방향 확정, 블럭 3 구현)**: 기록·구별·측정 결과는 **익명 오제로 아이디**(주소·비번 없이 자동 발급, observer_code 부여) 아래 저장한다. 익명 아이디 발급 시 **하루 1회 배달 시간**을 사용자가 정한다. 저장이 익명 아이디와 한 몸인 이유: 신원 없이 저장하면 본인에게 돌려줄(회수) 방법이 없다. 구현 전까지 화면엔 "저장되지 않았어요"를 정직하게 표시.
- 횟수 제한은 서버 판정: 비로그인 1일 1회, **오제로 아이디 보유자 1일 3회** (구현됨 2026-07-07 — 비밀 열쇠가 실제 관찰자일 때만. IP 해시+서버 날짜(UTC), measure_limits, fail-open, 실패한 측정은 안 셈). 초과 시: "오늘 측정은 여기까지예요. 내일 다시 비춰볼 수 있어요." — **입구 진입 시 사전 안내**(GET /api/measure), 구별을 다 시킨 뒤 문 닫지 않기.
- 언어: Accept-Language 자동 판별 — ko 외 전부 en (messages/en.json + 문구 배급기, API 오류 문구 포함. 구현됨 2026-07-05). 영어권 홈은 "Ozero's Mirror"가 메인, 한글 이름이 서브.
