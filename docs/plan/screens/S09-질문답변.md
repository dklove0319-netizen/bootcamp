# S09. 오늘의 질문 + 답변 (하루 루프 단계 8·9)

## 들어오는 길
- S08 감정 연결 → 다음 (망상 0개면 S07에서 바로)

## 화면 구성 (위에서 아래로)
1. 오늘의 질문 (AI 생성, 서버에서): "M월 D일에 이렇게 쓰셨어요 — '(원문)'. 오늘의 기록과 나란히 놓아보세요. (되묻는 질문)"
2. 답변 입력칸 (여러 줄)
3. 익명 공개 토글: "이 답변을 다른 관찰자에게 익명으로 보여줄까요" — 기본 꺼짐. 이 답변 하나에만 적용 (일괄 설정 없음)
4. 제출 버튼

## 규칙 (지시서 3-8·9, 6번)
- 질문은 반드시 누적 기록 기반 원문 인용 (요약·의역 금지). 인용할 문장이 없으면(첫날) 인용 없이 오늘 기록의 추상 표현을 되묻는다.
- 인용 원문·출처 날짜도 함께 저장 (question_quote_date·question_quote_text) — 구조화 저장 원칙.
- 토글을 켜면 그 순간의 시각·정책 버전을 함께 기록 (동의 b종 — 건별).
- 블럭 3에서는 오늘 기록 기반, 블럭 7에서 누적 기반으로 확장.

## 행동과 흐름
| 행동 | 성공하면 | 실패하면 |
|---|---|---|
| (진입 시) 질문 생성 요청 | 질문 표시 | AI 실패: 재시도 → 반복 실패 시 예비 고정 질문 "오늘의 기록에서 가장 오래 남는 문장은 무엇인가요." (루프를 끊지 않음) |
| 제출 | answer_text·answer_shared 저장 → S10 | "저장하지 못했어요. 작성한 내용은 그대로 있어요." + 재시도 |
