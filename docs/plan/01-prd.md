## 1. 개요

- 한 줄 정의: 21일간 매일 한 번, 사용자가 자기 하루를 기록하고 그 기록에서 사실과 자동해석("망상")을 스스로 구별하는 훈련 웹앱.
- 본질: 챗봇이 아니라 **판독과 회수**. 판독 = 기록에서 사실·망상·감정·행동을 분리해 저장. 회수 = 사용자의 과거 문장을 원문 그대로 날짜와 함께 되돌려주기.
- 목표 상태: 알아차림 → 선택 → 행동.
- 이 앱이 아닌 것: 감정 관리 앱 아님 · 위로 앱 아님 · 일기 앱 아님 · 부정적 생각을 긍정으로 바꾸는 앱은 더더욱 아님.
- 성공 기준: [미정 — 지시서에 정량 목표 없음. 단 측정 항목은 정의됨 → 6절 "측정 로깅"]

## 2. 문제·타겟

- 첫 사용자: "오제로 퀘스천 레터" 수료자. 방법론을 이미 아는 사람들이 먼저 쓴다. 신규 대중 아님.
- 사용 장면: 대부분 스마트폰으로 밤에 사용 → 모바일 우선.
- 초기 등록: 결제 없이 관리자 수동 등록으로 운영 (결제는 맨 마지막 빌드 단계).

## 3. 절대 원칙 (요약 — 전문은 지시서 2번, `.claude/CLAUDE.md` 상단에 상시 로드)

1. AI는 거울이지 분석가·선생·상담사가 아니다. 출력은 항상 "대조"와 "회수".
2. 사용자가 쓰지 않은 것을 지어내지 않는다. 과거 참조는 원문 인용 + 작성 날짜로만. 요약·의역·패턴 명명 금지.
3. 톤: 위로 금지 · 이모지 금지 · 감탄 금지 · 판단·충고·조언 금지 · 짧은 단정문 · ~이에요/~해요체.
4. 추상 언어는 바꿔주지 않고 되묻는다.
5. 행동의 주어는 사용자. AI 제안은 명시적 요청 시에만.
6. 격리(DB 층)는 예외 없이, 공유(제품 층)는 건별 선택으로만.
7. (사용자 추가 원칙, 2026-07-05) **사용자는 환자·내담자가 아니라 관찰자다.** 문제 있는 사람·고쳐야 하는 사람으로 보지 않는다. 화면 문구·설문 도구에 결핍·치료 어휘(증상·치료·회복·개선·극복 등) 금지, 능력 진술형 우선.
8. (사용자 추가 기준, 2026-07-05) **사실·망상 판별 기준** — 카메라 검사·근거 검사, 감정의 발생은 사실·해석은 망상, 성분 단위 분류, 방향 중립(긍정 망상도 동일 분류·교정 톤 금지), "망상" 용어 유지. 전문: `08-ai-prompts.md` 공통 판독 규칙 · `.claude/CLAUDE.md`.

### 21일 구조 (지시서 4번)

- 개인화 시작 — 기수제 아님. 1일차 = 첫 기록 제출일 (첫 기록 직전 S16 시작점 설문).
- 하루 1회 활성화 + 사용자 지정 고정 시각. 게이팅은 반드시 서버 판정. 기록 창 = 고정 시각 ~ 다음날 고정 시각 직전. 시간대 v1 한국 고정.
- 7·14일째 중간 거울 — 해석 없는 사실 제시 (예: 최다 반복 단어 3개).
- 14일째 시차 대면 — 1일째 답변 전문 그대로.
- 빠진 날 = 실패가 아니라 데이터. 복귀 오프닝: "어제는 기록이 없는 날입니다. 안 쓴 날은 어떤 날이었나요." 연속 기록 깨짐 표시 일절 금지.
- 21일 격자 달력 — 사실의 표시만. 배지·포인트·불꽃·연속기록 카운터 절대 금지.

### 21일 보고서 — 형식 확정 (2026-07-05 · 7개 구성)

1. 기록 사실 요약: 기록한 날 N일 / 기록 없는 날 M일 (평가 문구 없음)
2. 눈금 4종의 날짜별 추이 (표 또는 단순 선)
3. 가장 자주 이름 붙인 감정 + 가장 반복된 단어 (사실 제시만, 해석 없음)
4. 시차 대면 확장: 1일·7일·14일·21일 답변 전문을 날짜와 함께 나란히
5. 사용자가 직접 이은 망상·감정 연결 목록 (원문 그대로)
6. 시작점 / 21일째 설문(WHO-5·탈중심화 척도) 숫자 나란히 — 해석·평가 없음
7. 마지막 한 줄: 되묻는 질문 하나 (AI 생성, 원문 인용 기반)

### 다음 버전 로드맵 (v1 범위 밖 — 꿈은 층으로 보존, 2026-07-05 사용자 정보)

- **35일 심화 과정 — "거울 + 메스"**: 하루 루프 구조는 동일, 코스만 다름. 주차별 해부 주제 — 1주 나도 모르는 내 패턴 발견 / 2주 가족 패턴 / 3주 현재 관계 패턴 / 4주 돈 무의식 / 5주 의식·무의식 통합. AI가 먼저 한 꺼풀 벗겨주는 "메스" 모드 + 주 5일 리듬 (요일 규칙 확인 필요). 레터 가격 기준 189,000원.
  - **주의**: "메스" 모드는 절대 원칙 1(해석·분석 금지)의 **의도적 완화**다. 만들 때 코스별 원칙·프롬프트를 따로 정의해야 하며, 21일 거울 코스에 절대 새어 들어가면 안 된다.
- **졸업 이후 층 (35일 수료자 — "오제로 라이프스타일")**: 후기 근거는 `10-why-it-works.md`.
  1. **유지 모드 (관찰 일지)** — 21일 구조 없이 하루 루프(눈금·기록·구별·회수)만 남긴 도구. 판매 프레임은 "프로그램"이 아니라 "훈련은 끝났고 도구가 남는다" — 낮은 월 구독 후보. 수료자 후기 "오제로를 올리고 나서도 기록하고 싶은 일이 생기면 따로 정리해두었어요"가 이 수요의 증거
  2. **정기 재측정** — 분기마다 시작점 척도(WHO-5·탈중심화) 재측정. "관찰자의 눈이 얼마나 열려 있나" 정기 점검이 복귀 계기가 됨
  3. **졸업 관찰자 답변 풀** — 공개 동의된 졸업자 답변이 신입의 익명 답변 1개 풀에 들어가는 순환 (신입에게는 앞서 걸은 사람의 거울)
  - 댓글·피드·채팅은 졸업 층에서도 앱 안에 만들지 않는다 — 모임·커뮤니티는 앱 밖(레터·오프라인)에서
- **무의식 패턴 지도 (별도 프리미엄 상품 — 출시 후)**: 21일 거울 보고서(사실·원문 인용, 49,000원에 포함)와 구별되는 깊은 층. **원칙: AI는 추출·정렬·되묻기까지만 — 해석·라벨 생성 금지 ("당신의 무의식 문장은 X입니다" 같은 생성형 규정은 개인화된 라벨이라 더 위험). 깊은 거울(본인이 실제 쓴 문장의 반복을 날짜와 함께) + 인간 해석 층(운영자의 지각 — 자동화하지 않음, 이것이 해자)의 2단.** 첫 완주 기수 데이터로 운영자 수작업 프로토타입 → 소수 제공·인터뷰로 가격 발견 (앵커: 코스 49,000 위 · 35일 심화 189,000 아래). 35일 심화의 일부로 흡수될 수도 있음
- 창고 뼈대는 이미 준비됨: journeys(여정) 표의 course 값('mirror21'/'scalpel35'/추후 'practice')으로 코스를 구분 — **앱을 따로 만들지 않는다. 그리고 이 로드맵의 어떤 것도 v1에서 빌드하지 않는다** (지시서 11번).
