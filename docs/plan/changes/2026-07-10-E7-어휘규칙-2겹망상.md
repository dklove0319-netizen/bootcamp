# 변경 기록 — E-7 어휘 규칙 + 2겹 망상 해부 모듈 주입 (2026-07-10)

## ① 뭘 바꾸나 (한 줄)
운영자가 제공한 자세규정 모듈(어휘 규칙 LEXICON + 2겹 망상 해부)을 전 AI 창구 대본에 주입한다.

## ② 왜
- 운영자 규정 제공 (2026-07-10, 원문: docs/ozero_mirror_ai_자세규정_모듈-어휘-2겹망상.md) — 규정 문서 자체가 지시라 내용 승인 절차 불요, 본 문서는 변경 이력.
- 핵심: ⑴ 확인 불가능한 비유(사법·학교/측정·기계·치유영성·진단) 금지 — "아름다운 문장이 관찰을 대신하면 실패" ⑵ 2겹 망상(감정을 가진 자신을 서술한 문장)을 겹으로 나눠 카메라 검사를 2겹에 적용 ⑶ 닫는 질문 = 유도 금지·네/아니오 금지·예측 아니라 확인·이유 아니라 장면과 몸.

## ③ 영향
- craft.ts: CRAFT_COMMON(어휘 규칙·질문 닫기 규칙·반박 금지 ⑥), CRAFT_SPLIT(2겹 경계 규칙 + 모범례), CRAFT_QUESTION(자기판결 항목에 2겹 연결, 명중례 1개 추가) — 전 창구(측정·대조·질문·중간거울·보고서) 공통 반영.
- recall(어제의 거울) 창구: CRAFT_COMMON 미주입 상태였음 → 주입 추가.
- 충돌 해소 1건: E-5e 명중례 "그렇게 생각한 이유는 무엇인가요?"가 모듈의 '이유 금지'와 충돌 → "그 문장이 붙기 직전, 몸에서 무엇이 일어나고 있었나요?"(장면·몸)로 교체. (E-5e 채점 코멘트 6번의 취지는 '구어체'였고, 모듈이 각도를 정련한 것으로 해석.)
- 해석 2건: ⑴ '점수' — 눈금의 실제 수치(기분 7점)는 문자 그대로의 측정이라 예외(비유만 금지), 대본에 예외 명시 ⑵ '계약서' — 참가자 기록의 실물 지칭은 비유가 아니므로 허용.
- 화면 문구(ko/en): 감사 결과 금지 어휘 0건 — 변경 없음.
