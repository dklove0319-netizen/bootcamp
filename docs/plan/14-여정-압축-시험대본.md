# 여정 압축 시험 대본 — 운영자가 폰으로 직접 걷는 사람 검증 (2026-07-13)

> 검증 3층 중 비어 있던 "사람 여정" 층. 로봇은 버튼이 눌리는지를 보고, 사람은 말이 찌르는지를 본다.
> 준비물: 운영자 폰 (o000 열쇠가 있는 기기) + 30~40분. 구멍을 찾으면 화면 캡처 + "이거 이상한데" 한 줄로 클로드 창에 (현장 교정 루프).

## A코스 — 새 손님의 첫날 (약 15분, 시크릿 창)
1. 폰 브라우저 **시크릿 창**으로 ozero-mirror.vercel.app 열기 → 홈 장면 2줄("한 번 올라온 생각이…")이 내 마음을 잡는가?
2. [지금 비춰보기] → 진짜 오늘 장면 하나를 기록 → 사실/느낌/행동 칸 → [거울에 비춰보기]
3. 결과: 내 구별 vs 거울이 나란히 서는가? 거울의 이유가 정확한가(세탁·비유 없이)? 질문이 찌르는가? 계단 한 줄("오늘 것은 사진 한 장이에요")이 보이는가?
4. [오제로 아이디 받기] → 시각을 **일부러 2시로** 골라보기 → 새벽 안내가 뜨는가? → 21시로 바꾸고 발급
5. 발급 완료 화면에서 **[저녁 알림 켜기]** — 아이폰은 사파리 공유 → '홈 화면에 추가' 뒤에. (이러면 알림 구독 1호가 된다)
6. 질문에 답 남기기 → **다음날 그 시각에** 어제의 거울(질문+답+되비춤)이 돌아오는지 확인 — 이건 하루 뒤 숙제.

## B코스 — 21일을 30분에 (본 기기, o000 열쇠)
1. ozero-mirror.vercel.app**/seed** 열기 (운영자 전용 도구 — 주소 직접 입력)
2. 방법 둘 중 하나:
   - 빠르게: [21일치 심고 들어가보기] (준비된 시나리오)
   - 진짜같이: "내가 쓴 기록으로 심기"에 **사장님 진짜 기록**을 하루=한 덩어리, 빈 줄로 나눠 붙여넣고 [이 기록으로 21일치 심기] (거울이 하루씩 갈라 심어 몇 분 걸림)
3. 심은 뒤 순서대로: ① 오늘 루프(21일차 헤더·단계 줄) → ② 내 거울(격자) → ③ **나의 지도**(여러 날 돌아온 문장 — 감정이 달라도 문장이 같은 게 보이는가?) → ④ 21일 보고서(문답 2칸 작성 → 눈금 그래프·반복·돌아온 방식·시작점/21일째 문장·**다음 문**까지)
4. 다 보면 /seed로 돌아와 [내 원래 아이디로 돌아가기]

## C코스 — 결제 여정 (본 기기, 사장님 몫 ③과 동일)
1. 홈 → 내 거울 → 21일 코스 카드: 3단계 문구·레터 후기가 사고 싶게 만드는가?
2. [참여하기] → 검증 동의 체크 → [카드로 결제하기 (테스트)] — 돈 안 나감
3. 결제 후 /today 1일차: 시작점 문답(WHO-5 + "지금의 당신은 스스로를 어떤 사람이라고 생각하나요?") → 첫 루프 10단계 끝까지
4. 이날부터 매일 저녁 실사용 = 3층 검증이 상시로 돌기 시작

## 각 코스에서 특히 볼 것 (로봇이 못 보는 것)

- 질문을 읽고 **답을 쓰고 싶어지는가** (안 쓰고 싶으면 = 구멍)
- 기다림이 **답답한가** (AI 응답 몇 초의 체감)
- 오타·어색한 말투·이모지·위로 어휘 (발견 즉시 캡처)
