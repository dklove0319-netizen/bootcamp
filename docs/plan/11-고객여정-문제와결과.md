# 고객 여정 재설계 — 문제·결과 정렬 (2026-07-10, 근거: 금고 후기·추적기록 원문)

> 목적: "오제로의 거울이 누구의 어떤 문제를 풀고, 어떤 결과를 주는가"를 후기 실측으로 정의하고, 여정의 각 스테이지가 그 문제→결과 축 위에 서 있는지 점검한다.
> 근거 자료: docs/corpus/후기-1-퀘스천레터.md (참가자 원문) + 추적기록 7명 + 현행 화면 문구(ko.json)·PRICING_SPEC.

## 1. 사람들이 갖고 오는 문제 (후기 원문 근거 — 5개)

| # | 문제 | 참가자 원문 (금고) |
|---|---|---|
| 1 | **반추 — 생각이 멈추지 않아 에너지가 샌다** | "해석에 빠져 몇 시간을 곱씹는 등 반추하는데 큰 에너지를 들였다" / "그 감정에 심취해서 … 하루를 허비" |
| 2 | **감정 폭발이 관계를 상하게 한다** (아이·배우자·직장) | "소리를 지르지 않았다"(뒤집으면: 지르던 사람) / "화가나면 눈물부터 나오던 사람" |
| 3 | **타인의 마음 읽기에 휘둘린다** | "결국 알지못한 남들의 마음, 판단에 왜 집착하고 그것에 심하게 휘몰이 쳐졌나" / "외부 시선을 너무 신경쓰고" |
| 4 | **자기 판결 — 스스로를 부족한 사람으로 정해둔다** | "스스로를 부족한 사람, 실수하는 사람으로 바라보고 있었기에 그에 따른 망상들을 만들어내고 있었네요" |
| 5 | **같은 일이 반복된다** | "30년 가량 뭔가 늘 억울했고 … 비슷한 일들이 생긴다는 것을 깨닫고 환경이 아니라 내가 바뀌어야 함을 느꼈는데" |

**공통 뿌리 (제품의 문제 정의 한 줄):** 사실과 해석(망상)이 한 덩어리로 붙어 있어서, 해석이 곧 현실이 되어버리는 것. 구별이 안 되니 감정이 해석을 따라 날뛰고(1·2), 남의 속마음이 사실처럼 굳고(3), 자기 판결이 정체성이 되고(4), 같은 해석이 같은 상황을 재생산한다(5).

**표현 제약 (관점 원칙):** 문제를 말할 때도 진단·결핍 어휘 금지. 장면(카메라 언어)으로 말한다 — "당신은 반추가 심합니다" ✕ / "한 번 올라온 생각이 밤까지 이어지는 날" ○.

## 2. 21일 뒤 얻는 결과 (후기 실측 — 5개) 와 삶의 개선 영역

| # | 결과 | 참가자 원문 | 삶의 개선 |
|---|---|---|---|
| 1 | 구별 능력 (해상도) | "카메라의 화소가 올라간 느낌" / "라섹수술 한 기분" / "혼탁한 물을 흔들어 층을 분리" | 상황 파악 → 해결 순서가 생김 ("감정 인지→사실파악→해결책") |
| 2 | 선택의 틈 | "어떤 선택을 할건지 아주 잠깐이나마 생각하는 틈이 생겨가고" / "내가 선택권을 갖는 느낌" | 관계 — 소리 안 지름, 화 안 냄, 할 말은 함 |
| 3 | 에너지 회수 | "확실히 에너지가 덜 든다" / "마음의 너울이 그리 오래가지 않아요" | 시간 — 곱씹던 몇 시간이 돌아옴 ("그 시간에 자야해요") |
| 4 | 반복의 지도 | "아 또 반복이네" / "옛날 데이터를 복사해서 생각하고 있군" | 같은 상황의 재생산이 보임 → 끊을 자리가 보임 |
| 5 | 자기 신뢰 | "혼자 휘둘리지 않고 서있을 수 있는 힘" / 먼저 연락해 제안을 받음, 분리·이사를 스스로 결정 | 결정 — 남에게 "이게 맞아?" 묻지 않게 됨 |

**구조 (확정된 프레임):** 카타르시스 3단계 — ① 문장의 분리(즉시, 기록 1개) ② 반복의 발견(6일차~, 여러 날 필요) ③ 변화의 목격(15일~, 전/후 두 시점 필요). 결과 1은 ①에서, 4는 ②에서, 2·3·5는 ②③의 누적에서 나온다. **21일 = ②③이 물리적으로 가능한 최소 기간** (후기 실측: "6일차부터 부쩍 좋은 피드백이 많아짐").

## 3. 여정 스테이지 진단 — 지금 무엇을 말하고, 무엇이 비었나

| 스테이지 | 지금 화면이 말하는 것 (실제 문구) | 이 스테이지가 팔아야 하는 것 | 진단 |
|---|---|---|---|
| 홈 (입구) | "위로 대신, 정확하게 비춰드립니다" · "사실과 망상을 나누는 훈련이에요" | 방문자의 "내 얘기다" — **문제의 자기 인식** | ⚠️ 방법 언어만 있고 문제 언어(장면)가 없음. 문제→방법 다리를 방문자가 놓아야 함 |
| 무료 측정 | 구별 + 거울 대조 + 되묻는 질문 | ①단계 카타르시스 즉시 체험 | ✅ 강함 (E-5 정밀도). 단 "이건 사진 1장 — 21일이면 지도"라는 다음 계단 안내가 얇음 |
| 아이디 + 3일 미니 | "이 질문은 여기서 끝나지 않아요" — 내일 회수, 사흘 반복 되비춤 | 회수의 맛 (내 문장이 돌아온다) | ✅ 사다리 좋음 |
| 21일 판매 | "사흘이 이만큼이면, 21일은 반복의 지도가 됩니다" + 보고서 + 검증 동의 | ②③단계 이득의 **약속** — 기능이 아니라 결과 언어 | ⚠️ "반복의 지도·보고서" = 기능 언어. 결과(틈·에너지·자기 신뢰)와 "왜 21일인가"(3단계 근거)가 없음 |
| 루프 1~7일 | 어제의 거울 회수 + 단계 한 줄("가르는 주") | 이득 체감 전 구간을 견디게 하기 | ✅ E-6로 보강됨. 이탈은 step_events 데이터 쌓인 뒤 재점검 |
| 루프 8~14일 | 중간 거울(7·14) — 반복 구조 반사 | ②단계 카타르시스 | ✅ 있음 (E-4·E-5) |
| 루프 15~21일 | 단계 한 줄("나란히 놓는 주") + 행동 데이터 | ③단계 준비 | ✅ 있음 |
| 21일 보고서 | 전후 문장·돌아온 방식·반복의 지도·WHO-5 전후·마지막 질문 | ③단계 카타르시스 — 변화의 증거 회수 | ✅ E-6로 완성 |
| **종료 후** | (없음 — 보고서에서 "내 거울 보기"로 끝) | **다음 문** — 훈련의 지속 또는 심화 | ❌ **비어 있음.** 후기: "오제로 레터 없는 일상이 상상이 안 되는데" / "한번 더 무의식의 해부로 초대 주세요" — 가장 뜨거운 순간에 갈 곳이 없다 |

## 4. 재설계 방향 (스테이지별 한 줄)

1. **홈**: 방법 선언 앞에 문제 장면 1~2줄 (카메라 언어, 진단 어휘 0) → "그 생각, 사실인가요?"가 자연히 이어지게.
2. **측정 결과**: 결과 하단에 계단 한 줄 — "오늘 것은 사진 한 장이에요. 같은 자리에서 21장을 찍으면, 반복이 보여요."
3. **판매**: 3단계 구조로 약속을 다시 씀 — 1주 구별 / 2주 반복 발견 / 3주 전후 대조, 그리고 결과 언어(후기 실측 5개 중 2·3·5). "왜 21일인가"에 답이 있게.
4. **루프·보고서**: 현행 유지 (E-4~E-6로 이미 축 위에 있음).
5. **종료 후**: 보고서 끝에 "다음 문" 1개 — 운영자 심화(레터·해부)로의 다리 또는 2회차. **무엇을 놓을지는 운영자 결정.**

## 5. 빌드 후보 (승인 대기)

- [ ] J-1. 입구·측정 결과·판매 문구 재작업 (위 1·2·3) — 카피 톤 스펙(짧은 평서문·~이에요체·위로 금지) + 관점 원칙(결핍 어휘 0) 준수. 확인: 전 문구 한/영, 금지 어휘 검사.
- [ ] J-2. 보고서 끝 "다음 문" — 운영자 결정(심화 안내 vs 2회차 vs 보류) 후.
- [ ] J-3. (보류) 1~7일 이탈 보강 — step_events 실데이터가 쌓인 뒤 근거 있게.

## 6. 운영자만 정할 수 있는 것 2개

1. **종료 후 다리**: 21일을 마친 사람에게 무엇을 여나 — (a) 퀘스천 레터·무의식 해부(운영자 심화, 앱 밖) 안내 (b) 2회차 코스 (c) v1은 보류. 후기 근거는 (a)를 가리킨다.
2. **판매 페이지의 사회적 증거**: 레터 후기를 앱 판매에 쓰려면 "퀘스천 레터 참가자 후기"로 출처를 정직하게 밝혀야 함 (거울 자체 후기는 아직 없음). E-6의 21일째 문답이 쌓이면 거울 자체 후기로 교체 가능 — 공유 동의 장치는 별도 논의로 보류 중.
