## 한 문장 소개
나는 **오제로 퀘스천 레터 수료자와 신규 대중**을 위해 **21일 사실·망상 구별 훈련 웹앱 "오제로의 거울"** 을, 핵심 기능 **무료 즉시 측정 + 하루 10단계 기록 루프(판독과 회수)** 로 만든다.

## 데이터 (뭘 저장하나) → 상세 docs/plan/05-db.md
- 표: **profiles** (관찰자 — 익명 코드 o###, 기록 시각) · **journeys** (여정 — 코스·시작일. "새 21일"과 추후 35일 심화가 들어올 자리)
- 표: **daily_entries** (하루 기록 — 상태 숫자 4종·자유 기록·내 구별·AI 대조·망상감정 연결·질문(인용 출처 포함)·답변(건별 공개)·행동·실행 여부, 전부 별도 필드 = 구조화 저장)
- 표: **consents** (동의 이력 — 종류·버전·시각·철회) · **step_events** (단계별 이탈 로그) · **assessments** (시작점·21일째 설문)
- 관계: 한 profiles 가 여러 journeys, 한 journeys 가 여러 daily_entries → 1:N (꼬리표, FK 제약 없음)
